This project provides basic functionality of a Virtual Vending Machine.

You can interrogate the API to test basic functionality as follows:


Get available products (stubbed data, loaded on startup)

http://localhost:61839/api/VendingMachine/getAvailableProducts


Get availability of a product

http://localhost:61839/api/VendingMachine/getProductByName?strName=Product1


Add some credit to a given user

http://localhost:61839/api/User/addRemoveUserCredit?user=User1&money=100


Check their balance

http://localhost:61839/api/User/GetUserBalance?user=User1


Purchase some products

http://localhost:61839/api/VendingMachine/purchaseProduct?user=User1&productName=Product1&quantity=1

http://localhost:61839/api/VendingMachine/purchaseProduct?user=User1&productName=Product2&quantity=1


Check purchase of a given user

http://localhost:61839/api/User/GetUserPurchases?user=User1


Get the remaining credit back

http://localhost:61839/api/User/GetUserCreditBack?user=User1
